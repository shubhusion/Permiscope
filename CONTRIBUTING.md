# Contributing to Permiscope

First off, thank you for considering contributing to Permiscope! It's people like you that make Permiscope such a great tool for the AI community.

## Code of Conduct

Help us keep Permiscope open and inclusive. Please read and follow our [Code of Conduct](CODE_OF_CONDUCT.md) (coming soon).

## How Can I Contribute?

### Reporting Bugs
- Use GitHub Issues to report bugs.
- Provide a clear, descriptive title and as much detail as possible.

### Suggesting Enhancements
- Enhancement suggestions are tracked as GitHub issues.
- Describe the feature and why it would be useful.

### Pull Requests
- Fork the repository.
- Create a new branch for your feature or bug fix.
- Ensure your code follows the project's linting and formatting rules.
- Run tests before submitting.
- Provide a clear description of the change in your PR.

## Development Setup

1.  Clone the repo: `git clone https://github.com/shubhusion/permiscope.git`
2.  Install dependencies: `npm install`
3.  Build: `npm run build`
4.  Test: `npm test`

## Questions?
Reach out to the maintainers via GitHub Issues.
